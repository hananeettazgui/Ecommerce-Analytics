# Olist E-commerce Analytics Project

This project demonstrates my work on analyzing the **Olist Brazilian e-commerce dataset**.  
It focuses on **data cleaning, KPI calculation, and interactive dashboards** to support business decision-making.  

---

## Project Overview

**Olist** is a Brazilian marketplace connecting small and medium enterprises  to millions of customers across the country.  
It centralizes e-commerce operations such as orders, payments, logistics, and customer reviews.  
This dataset covers **2016–2018** and includes nearly 99,000 orders, providing a rich view of e-commerce activity in Brazil.  

The goal of this project is to transform raw data into **actionable insights** to help improve:

- Commercial and financial performance  
- Customer satisfaction and retention  
- Logistic efficiency and delivery reliability  
- Seller performance evaluation  

---

## Project Scope

**Included:**

- Cleaning, structuring, and modeling Olist data in a relational database (Snowflake / SQL)  
- Calculating KPIs across multiple dimensions: financial, customer, product, logistics, and satisfaction  
- Creating interactive dashboards in Power BI  
- Representing the relational data model (schema)  


---

## Project Workflow

1. **Data Modeling (SQL / Snowflake)**  
   - Imported all tables: customers, orders, order_items, products, payments, reviews, delivery  
   - Defined primary and foreign keys for data integrity  
   - Built SQL joins and views to simplify analysis

2. **Data Cleaning and Preparation**  
   - Handled missing values based on business logic  
   - Structured and prepared data for dashboards and analysis

3. **KPI Calculation & Analysis**  
   - Financial performance: revenue, average basket, sales trends  
   - Customer behavior: segmentation, retention, geographic distribution  
   - Product and category performance  
   - Logistics: delivery time, delays, reliability  
   - Customer satisfaction: ratings and reviews  

4. **Dashboard Development (Power BI)**  
   - Created interactive dashboards with filters by period, region, and product category  
   - Dashboards provide a clear, visual overview for decision-making
